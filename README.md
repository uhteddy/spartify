# Spartify
Spartify is a client that allows you to easily create spotify parties that allow your guests to request songs, vote on songs, and more!

### FAQ
**Q: Why use this over Spotify Jams?**

Spotify Jams are great for small parties, car rides, and other small gatherings. However, they are not ideal for larger parties or events where you want more control over the music. Spartify allows you to create a more personalized experience for your guests, with features like song requests, voting, and more.

**Q: Is Spartify free to use?**

Spartify is free to use, but it does require a Spotify Premium account to function. This is because Spartify uses the Spotify API to control playback and manage playlists.

**Q: How do I create a party?**
To create a party, simply click on the "Create Party" button and follow the prompts to set up your party. You can choose a name for your party, set a password (optional), and customize other settings.

**Q: How do guests join my party?**
Guests can join your party by entering the party URL that is generated when you create the party or scanning the QR code generated.
